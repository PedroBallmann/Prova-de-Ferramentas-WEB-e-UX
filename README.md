# Lumina Tech Solar - Landing Page

Projeto de landing page one-page para a startup ficticia **Lumina Tech Solar**, desenvolvido com HTML5 e CSS3.

## Integrantes

- Pedro Ballmann
- Gustavo Silvestre
- Lucas Kiem
- Leonardo Leite

## Como visualizar o projeto

1. Baixe ou clone este repositorio.
2. Abra a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

Nao e necessario instalar dependencias, rodar servidor, executar build ou configurar ferramentas extras. Ao abrir o `index.html`, o navegador carrega o texto, os estilos e a imagem local automaticamente.

## Arquivos principais

- `index.html`: estrutura semantica da pagina.
- `styles.css`: estilos, responsividade, cores, tipografia e estados de interacao.
- `assets/solar-dashboard.svg`: imagem local usada na secao hero.

## Requisitos atendidos

- Secao hero com proposta de valor clara.
- Menu fixo com ancoras internas.
- Cards de beneficios organizados com CSS Grid/Flexbox.
- Layout responsivo para desktop e mobile.
- Secao de prova social com depoimentos.
- Formulario de contato com labels visiveis.
- Uso de tags semanticas como `header`, `nav`, `main`, `section`, `article`, `form` e `footer`.
- Imagem com atributo `alt`.
- CSS organizado com variaveis e media queries.

## Tecnologias

- HTML5
- CSS3
